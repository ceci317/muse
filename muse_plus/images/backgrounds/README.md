# 🎨 Muse Plus 背景图片指南

## 📁 图片放置说明

请将你的5张浪漫背景图片按以下命名规则放在此目录中：

- `bg1.jpg` - 第一张浪漫图片
- `bg2.jpg` - 第二张浪漫图片  
- `bg3.jpg` - 第三张浪漫图片
- `bg4.jpg` - 第四张浪漫图片
- `bg5.jpg` - 第五张浪漫图片

## 🎯 图片要求

- **格式**: JPG 或 PNG
- **尺寸**: 建议 1920x1080 或更高分辨率
- **内容**: 浪漫、暧昧主题的图片
- **色调**: 温暖色调效果更佳

## 💡 使用说明

1. 将图片文件拖拽到此目录
2. 确保文件名完全匹配（bg1.jpg, bg2.jpg, etc.）
3. 刷新浏览器页面即可看到效果

## 🌟 备用方案

如果图片无法加载，系统会自动显示美丽的浪漫主题渐变背景作为备用方案。

## 🔧 故障排除

如果图片不显示：
1. 检查文件名是否正确
2. 确认图片格式为 JPG 或 PNG
3. 查看浏览器控制台的错误信息
4. 尝试刷新页面
