## 🚀 快速开始

### 方法1: 使用你自己的图片（推荐）
1. **准备图片**: 将你的5张浪漫图片重命名为：
   - `bg1.jpg` - 第一张浪漫图片
   - `bg2.jpg` - 第二张浪漫图片
   - `bg3.jpg` - 第三张浪漫图片
   - `bg4.jpg` - 第四张浪漫图片
   - `bg5.jpg` - 第五张浪漫图片

2. **放置图片**: 将这些文件放入 `images/backgrounds/` 目录

3. **检查状态**: 运行检查脚本
   ```bash
   ./add-images.sh
   ```

4. **测试效果**: 访问 http://localhost:3000/test-images.html 查看预览

5. **查看主页**: 访问 http://localhost:3000/ 查看轮播效果

### 方法2: 使用示例图片（测试用）
1. **生成示例**: 访问 http://localhost:3000/images/backgrounds/generate-samples.html
2. **保存图片**: 右键点击每个图片 → 另存为 → 保存为对应文件名
3. **放入目录**: 将保存的图片放入 `images/backgrounds/` 目录

## 🎯 图片要求

- **格式**: JPG 或 PNG
- **尺寸**: 建议 1920x1080 或更高分辨率
- **内容**: 浪漫、暧昧主题的图片
- **色调**: 温暖色调效果更佳
- **文件大小**: 建议每张图片不超过5MB

## 🔍 故障排除

### 图片不显示？
1. 检查文件名是否完全匹配（bg1.jpg, bg2.jpg, etc.）
2. 确认图片格式为 JPG 或 PNG
3. 查看浏览器控制台的错误信息
4. 运行 `./add-images.sh` 检查状态

### 切换太慢？
- 当前设置为4秒切换，2-3秒过渡
- 如需调整，可修改 `index.html` 中的时间参数

### 图片模糊？
- 确保图片分辨率足够高（建议1920x1080+）
- 检查图片质量设置
- 避免过度压缩的图片
